# Security Policy

## Reporting a Vulnerability
Please report security vulnerabilities to **[security@softics.de](mailto:security@softics.de)**.

Include:
  - A description of the issue
  - Steps to reproduce
  - Affected versions (if known)
  - Your contact information for follow-up

We aim to acknowledge reports within 2 business days and to provide
a full response within 14 days.

Please **do not** open public issues for security-sensitive matters.

## Supported Versions

Security fixes are provided for the latest released version of each
project. Older versions may receive fixes at our discretion.

## Scope

This policy covers all repositories within the
[Softics-GmbH](https://github.com/Softics-GmbH) GitHub organization.
